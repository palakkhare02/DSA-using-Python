What is a Dynamic Array?

A Dynamic Array is an array that can automatically grow or shrink in size during runtime.

In normal (static) arrays:

Size is fixed.

You cannot add more elements once it is full.

In dynamic arrays:

Size changes automatically.

When the array becomes full, a new bigger array is created and old elements are copied into it.

Example:

Start with size = 4

Add 5th element ‚Üí array resizes to size = 8

This is how languages like Python (list), Java (ArrayList), and C++ (vector) work internally.

Create an initial array with small capacity.

Create a new array with double size.

Copy all old elements to the new array.

Continue inserting.

This resizing takes O(n) time, but since it does not happen often, the average insertion time is O(1) (amortized).
